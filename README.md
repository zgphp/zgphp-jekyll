## How to get a Vimeo video thumbnail

Visit:
```
http://vimeo.com/api/oembed.json?url=http://player.vimeo.com/video/VIDEO_ID
```

For example:
```
http://vimeo.com/api/oembed.json?url=http://player.vimeo.com/video/66249603
```
